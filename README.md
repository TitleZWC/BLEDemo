# BLEDemo
a Demo to use the BLE library:https://github.com/TitleZWC/BlueToothLowEnergyLibrary.git

这是一个使用一个蓝牙library的例子，本例子使用蓝牙通过串口调试助手与手机进行了数据的交互。

运行本Demo，打开应用以后：<br>

选择允许后，就可以打开蓝牙<br>
<img src="/pic/Screenshot_2016-01-28-18-31-04.png" alt="Screenshot_2016-01-28-18-31-04.png" title="Screenshot_2016-01-28-18-31-04.png" width="400" height="660" />

然后选择菜单，进行搜索<br>
<img src="/pic/Screenshot_2016-01-28-18-31-10.png" alt="Screenshot_2016-01-28-18-31-10.png" title="Screenshot_2016-01-28-18-31-10.png" width="400" height="660" />

搜索结果如图所示：<br>

<img src="/pic/Screenshot_2016-01-28-18-31-38.png" alt="Screenshot_2016-01-28-18-31-38.png" title="Screenshot_2016-01-28-18-31-38.png" width="400" height="660" />
